## Portfolio Desktop Overview

This project is a desktop-like portfolio website for Mustafa Jamshidi. It features:

- **Dark theme** and custom background for a modern look.
- **Draggable desktop icons** for GitHub, LinkedIn, Email, Experience, Projects, Resume, Education, Skills, and a TicTacToe game.
- **Double-click icons** to open popups (Experience, Education, Projects, Skills) or external links (GitHub, LinkedIn, Email, Resume PDF).
- **TicTacToe game**: Playable in a popup by double-clicking the game icon.
- **Live time and date** displayed at the top of the page.
- **Responsive design** for desktop and mobile.

All code is in plain HTML, CSS, and JavaScript. Icons and popups are styled for a Windows-like experience. Drag-and-drop is smooth and prevents accidental movement. Popups can be closed by clicking outside or using the close button.
